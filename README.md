[ThinkGO](https://www.think-ts.cn) 是一个轻量级的GO WEB应用框架，整合了各种常用SDK以及企业级常用的技术方案，为减少了开发人员的学习成本，提高团队的开发效率而生。

## 安装

通过脚手架安装

```
npm install think-ts-cli -g

think-ts-cli init demoApp
```

直接克隆

```
git clone https://github.com/zy598586050/think-go.git
```

安装依赖

```
go mod tidy
```

启动项目

```
go run main.go
```
